GRID LAYOUT

Day11 was focused on grid-template rows and grid-tempelate-rows. These are used to specify the number of rows and columns of your grid layout by specifying their width.

Today we will be focusing on

grid lines:
this are the lines between rows and columns with each bearing their respective names.A grid item can be placed between grid lines using grid-column and grid-rows.the start and end can also be defined seperately by adding "-start and -end".if the end starts span it specifies how many rows or columns it will span.
grid-areas:
Short-hand for grid columns and rows
grid-gaps
uses the gaps property It specifies the space in between the columns and row. To define a specific property, the column or row is added before the gap with an "-"
