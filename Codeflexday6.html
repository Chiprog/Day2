STYLES

Styles are used to beautify a page. They can be veiwed as the muscles of a page. In the absence of a style, a webpage will look boring and unnatractive

styles in html are mainly done using CSS

css styles in a page are grouped ino

  1. inline css
  2. external css
  3. internal css

INLINE CSS:

This is used to style a particular element. The style is applied within the opening tag of the element using the style attribute.

INTERNAL CSS:

This is used to style only one html page. The styling is done within the page. To perform internal css, the style tag is used in the head of the element.

EXTERNAL CSS:

This form of styling stores the css style in anothe file. The file has the extension .css and is acessed using the link tag in the head element. One css file can be used to style multiple pages
